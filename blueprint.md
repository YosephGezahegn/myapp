# Project Blueprint - Confetti App

## Overview
A simple Flutter application that demonstrates a confetti effect triggered by a button click, with a vertical slider to control the intensity.

## Features implemented
- Basic Flutter counter app structure.
- Integration of the `confetti` package.
- Confetti effect that triggers when the floating action button is pressed.
- Confetti particles fall from the top of the screen.
- Vertical slider to control the "confetti level" (number of particles).
- Modern Material 3 design with a harmonious color scheme.
- Responsive layout for Web and Android.
- Android label set to "Confetti App".

## Current Plan
- **Build APK**: Generate a release APK for the user to download and install on their Android device.
- **Command**: `flutter build apk --release`
- **Output**: The APK will be located in the `build/app/outputs/flutter-apk/` directory.
